# 에이전틱 (Agentic)이란?\n\n### 사전적 정의\n에이전틱(Agentic)은 인공지능 시스템이 단순히 정보를 조회하거나 텍스트를 생성하는 단계에서 나아가, 부여받은 목표를 달성하기 위해 스스로 판단하고 외부 도구(Tools)나 API를 활용하여 실질적인 과업(Action)을 수행하는 자율적 성질을 의미합니다. 이는 시스템이 독립적인 행위자(Agent)로서 기능할 수 있는 능력을 강조하는 개념입니다.\n\n### 실무 사용 예시\n전통적인 챗봇이 배송 규정을 안내하는 수준에 그친다면, 에이전틱 AI는 사용자의 요청에 따라 실시간 물류 데이터베이스에 접속하여 배송 상태를 확인하고 직접 반품 프로세스를 실행하는 등 실질적인 업무를 완결합니다.\n\n### 관련 단어\n- AI 에이전트 (AI Agent)\n- 도구 호출 (Tool Calling)\n- MCP (Model Context Protocol)
