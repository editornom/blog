# 什么是 Agentic (智能体化)？

### 词典定义
Agentic (智能体化) 是指人工智能系统不再局限于简单的信息查询或文本生成，而是为了达成既定目标，能够自主做出判断并利用外部工具 (Tools) 或 API 执行实际任务 (Action) 的自治属性。这一概念强调了系统作为独立行为体 (Agent) 发挥作用的能力。

### 实务应用案例
如果说传统的聊天机器人仅停留在提供配送规则指南的层面，那么 Agentic AI 则可以根据用户的请求，实时访问物流数据库核对配送状态，并直接执行退货流程，从而完成实质性的工作闭环。

### 相关词汇
- AI Agent (人工智能智能体)
- Tool Calling (工具调用)
- MCP (Model Context Protocol)
