# エージェンティック（Agentic）とは？

### 辞書的な定義
エージェンティック（Agentic）とは、AIシステムが単に情報を照会したりテキストを生成したりする段階を超え、与えられた目標を達成するために自ら判断し、外部ツール（Tools）やAPIを活用して実質的なタスク（Action）を遂行する自律的な性質を意味します。これは、システムが独立した行為者（Agent）として機能できる能力を強調する概念です。

### 実務における活用事例
従来のチャットボットが配送規定を案内するレベルにとどまるのに対し、エージェンティック AIは、ユーザーのリクエストに応じてリアルタイムの物流データベースにアクセスして配送状況を確認し、直接返品プロセスを実行するなど、実質的な業務を完結させます。
